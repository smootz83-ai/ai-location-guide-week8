# 📍 나만의 위치 안내 서비스

내 주변의 장소를 쉽고 빠르게 찾아보는 위치 안내 웹 애플리케이션입니다.

## 🛠 기술 스택

- **Node.js** v24+
- **Express** v4
- **HTML / CSS** (Vanilla)

## 📂 프로젝트 구조

```
AI_위치안내_8주차/
├── server.js          # Express 서버 (포트 3000)
├── package.json       # 프로젝트 설정 및 의존성
├── .gitignore         # Git 제외 파일 목록
└── public/
    ├── index.html     # 메인 웹페이지
    └── style.css      # 스타일시트
```

## 🚀 실행 방법

### 1. 저장소 클론
```bash
git clone git@github.com:smootz83-ai/ai-location-guide-week8.git
cd ai-location-guide-week8
```

### 2. 패키지 설치
```bash
npm install
```

### 3. 서버 실행
```bash
npm start
```

### 4. 브라우저에서 확인
```
http://localhost:3000
```
